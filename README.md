# CircuitDetector

## How to run :)

Run in `CircuitDetector/src/`:

```bash
python single_pallet_run {pallet_number}

Example: $python single_pallet_run 24771
```
This generates the CSVs and provides evaluation metrics for 1 pallet.

Run this to generate all CSVs and evaluation metrics across the full dataset:
```
final_eval.py
```
